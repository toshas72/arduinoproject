# arduino project

Smart Turn Signal System


Our project is about to improve drivers behaviour by fixing their mistakes, 
by detecting sharp steering movements and automatically activates the left or
right turn signal if the driver forgets to use it.


Our goals:

1. Improve Road Safety
2. Raise Drivers Avareness
3. Prevent Car Accidents


Problems we have now:

Many drivers forget to use their turn signals.
Sudden steering changes can cause accidents.
Lack of awareness when changing lanes or turning.
No simple low-cost system that assists drivers automatically.


Our Solution:

Create and manufacture simple system using Arduino that detects steering direction 
and automatically activates the correct turn signal. This system helps drivers 
signal properly even if they forget.


How it works:

1. Sensor which detecting steering aceleration.
2. Leds simulating turning signals.
3. LCD display showing messages about signals or telling that it turned signals automaticly.
4. Arduino, which will process all the data from sencors.
